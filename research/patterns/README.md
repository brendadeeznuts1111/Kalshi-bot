# Pattern reports (static extraction)

Generated by `bun run agent patterns` — reads GitHub file contents for detector evidence paths (via `gh` API, no clone).

| Output | Purpose |
|--------|---------|
| `patterns-latest-{dimension}.md` | Human-readable pattern report |
| `patterns-latest-{dimension}.json` | Structured hits for tooling / Phase 3 |

Categories include `auth`, `orders`, `loop`, `errors`, `structure`, `tests`, and **`bunFeatures`** (Bun.serve, bun:sqlite, Bun.cron, etc.).

Architecture summary: `bun run agent blueprint` → `research/reports/architecture-blueprint.md`.

Re-run after dimension research updates or when refining heuristics in `src/agent/pattern-extract.ts`.
